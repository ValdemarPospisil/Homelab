# Product Definition - Homelab

## Initial Concept
hey this is a homelab setup that i use on my server which is a notebook with 8gb ram and some old nvidia graphics card. I use this for entertainment but also to learn more about devops which is my ideal career path

## Target Audience
- **Primary User:** The owner (sole user), focused on personal media consumption and professional skill development in DevOps.

## Core Goals
- **Entertainment:** Provide a reliable media ecosystem (Plex, Audiobookshelf, Kavita).
- **DevOps Learning:** Serve as a practical lab for mastering monitoring, infrastructure as code, and CI/CD.
- **Balanced Performance:** Maintain a stable environment for media while allowing for significant experimentation with DevOps tools within 8GB RAM constraints.

## Key Features & DevOps Focus
- **Media Suite:** Plex for video, Audiobookshelf for audiobooks/podcasts, and Kavita for books/manga.
- **Monitoring & Observability:** Full stack implementation using Prometheus, Grafana, and Node Exporter to track system health and resource usage.
- **Infrastructure as Code (IaC):** Automation of host configuration and container orchestration using Ansible.
- **Continuous Integration/Deployment (CI/CD):** GitOps workflow implemented to automate deployments and host updates via Ansible and Docker Compose.
- **Network Management:** Ad-blocking and DNS management via AdGuard Home.
